# SMKIT Baitul Aziz V2

Website resmi untuk SMK IT Baitul Aziz.

## Fitur

- Informasi sekolah
- Berita & pengumuman
- Galeri foto
- PPDB
- Kontak
- Admin Dashboard

## Instalasi

1. Clone repositori:
    ```bash
    git clone https://github.com/username/SMKITBaitulAziz_V2.git
    ```
2. Masuk ke folder proyek:
    ```bash
    cd SMKITBaitulAziz_V2
    ```
3. Ikuti instruksi instalasi sesuai dokumentasi proyek.
